Fetch

  • 1.
    Fetchfn

A set of functions for working with maps.

Many functions for maps, which implement the Enumerable protocol, are found in the Enum module. Additionally, the following functions for maps are found in Kernel:

Example

1import { fetch } from '@brod/fetch'
2import { Json } from '@brod/json'
3
4const url = 'https://jsonplaceholder.typicode.com/todos/1'
5const response = fetch(Json, url, ({ data }) => ({
6     id: data['id'].toInteger(),
7     title: data['title'].toText(),
8     completed: data['completed'].toBoolean()
9}))
10response.data === {
11     id: 1,
12     title: 'quis ut nam facilis et officia qui',
13     completed: true
14}
15// => true

Interfaces

Fetchfn

▸ FetchFn(type, url): AwaitableFetchRequest

Fetch data at url as JSON

Example

1const response = fetch(Json, 'https://example.com/posts/1')
2response.status
3// => 200

▸ FetchFn(type, init): AwaitableFetchRequest

Fetch data at url using request as object properties

Example

1const response = fetch(Json, {
2     method: 'get'
3     url: 'https://example.com/posts/1',
4})
5response.status
6// => 200

▸ FetchFn(type, init, callback): AwaitableFetchRequest

Fetch data at url as JSON and parse response in callback

Example

1const response = fetch(
2     Json,
3     'https://example.com/posts/1',
4     response => ({
5         title: response.data['title'].toString(),
6         content: response.data['content'].toString(),
7         postedAt: response.data['date'].toDate()
8     })
9)
10response.title
11// => 'my post'
